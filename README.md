# Integra Browser

Стильный браузер для Windows на основе Chromium — в дизайне [Initial](https://github.com/saikodev-ru/initial-web).

## Особенности

- **Chromium-движок** — через Electron, без сторонних модификаций
- **Без телеметрии** — отключены crash reporting, safe browsing, background networking
- **Обход DPI/ТСПУ** — встроенная поддержка [zapret (winws)](https://github.com/bol-van/zapret) и [GoodbyeDPI](https://github.com/ValdikSS/GoodbyeDPI)
- **Дизайн Initial** — graphite palette, lavender→teal accent, Google Sans
- **Закладки** — добавление, удаление, панель закладок, управление
- **Настройки** — поисковая система, тема, размер шрифта, экспорт/импорт
- **Drag & Drop вкладок** — перетаскивание вкладок для изменения порядка
- **Панель закладок** — быстрый доступ к любимым сайтам прямо в интерфейсе

## Быстрый старт

```bash
npm install
npm start
```

## Сборка для Windows

```bash
npm run build:win
# → dist/Integra Setup x.x.x.exe
```

## Обход DPI/ТСПУ

Положи бинарник в папку `bypass/` (подробнее в [bypass/README.md](bypass/README.md)):

| Файл | Источник |
|------|----------|
| `winws.exe` + `WinDivert64.dll` | [zapret releases](https://github.com/bol-van/zapret/releases) |
| `goodbyedpi.exe` | [GoodbyeDPI releases](https://github.com/ValdikSS/GoodbyeDPI/releases) |

Кнопка "Обход" появится автоматически при наличии бинарника.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+T` | Новая вкладка |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+L` | Адресная строка |
| `Ctrl+R` / `F5` | Обновить |
| `Ctrl+1..9` | Переключить вкладку |
| `Ctrl+D` | Добавить/убрать закладку |
| `Ctrl+B` | Показать/скрыть панель закладок |
| `Ctrl+,` | Открыть настройки |
| `Alt+←` | Назад |
| `Alt+→` | Вперёд |

## Настройки

Доступны через кнопку ⚙ в панели навигации или `Ctrl+,`:

- **Поисковая система** — Яндекс, Google, DuckDuckGo
- **Тема оформления** — тёмная, системная
- **Панель закладок** — показать/скрыть
- **Размер шрифта** — от 12px до 18px
- **Обход DPI при запуске** — автозапуск winws/GoodbyeDPI
- **Экспорт/Импорт настроек** — резервное копирование
- **Сброс настроек** — возврат к значениям по умолчанию

## Закладки

- Нажмите **★** в адресной строке или `Ctrl+D` для добавления текущей страницы
- Закладки отображаются в панели закладок под адресной строкой
- Нажмите **⋯** на панели закладок для полного списка
- На новой вкладке ваши закладки показываются как плитки
- Поддерживается поиск по закладкам

## Drag & Drop

Перетаскивайте вкладки для изменения их порядка — просто grab and drop.

## Требования

- Node.js 18+
- Windows 10/11 (для bypass — права администратора)
